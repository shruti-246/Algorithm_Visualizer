# Algorithm Visualizer

An interactive web application for learning algorithms through **visualization, animation, and pseudocode tracking**.

Users can build custom binary trees and watch algorithms execute **step-by-step**, making it easier to understand how they work internally.

**Live Demo:**  
https://visual-algos.vercel.app

---

## Features

### Sorting Algorithm Visualizations

The visualizer includes several sorting algorithms:

- **Bubble Sort**
- **Insertion Sort**
- **Quick Sort**
- **Merge Sort**

Each sorting visualization shows:

- Comparisons between elements
- Swap operations
- Step-by-step algorithm execution
- Pseudocode highlighting synchronized with the animation

---

### Tree Traversal Visualizations

Users can build their own binary tree and visualize traversal algorithms.

Supported traversals:

- **Breadth-First Search (BFS)**
- **Depth-First Search (DFS)**

Features include:

- Interactive binary tree builder
- Queue visualization for BFS
- Stack visualization for DFS
- Active traversal edge highlighting
- Node visitation tracking

---

### String Matching Visualization

The project also includes visualizations for string matching algorithms.

Features:

- Pattern searching in text
- Step-by-step matching process
- Visual feedback for matches and mismatches
- Algorithm progress tracking

---

### Interactive Tree Builder

Users can create their own binary trees directly in the interface.

- Right-click a node to add left or right children
- Edit node values dynamically
- Automatic tree layout with visual layers

---

### Algorithm Insights

The interface also displays:

- Traversal order
- Visited node count
- Node and edge metrics
- Pseudocode highlighting during execution

## Prerequisites

Before running the project locally, make sure you have the following installed:

- **Node.js** (version 18 or later recommended)
- **npm** (comes with Node.js)

## Run Locally

```bash
# Clone the repository
git clone https://github.com/shruti-246/Algorithm_Visualizer.git

# Navigate to the project directory
cd Algorithm_Visualizer

# Install dependencies
npm install

# Start development server
npm run dev
```
